# Excel Data Analytics Project – 21 Day Challenge

## 📌 Problem Statement
The objective of this project is to analyze and transform raw data into meaningful insights using Microsoft Excel. The project focuses on understanding data patterns, improving data quality, and presenting insights through an interactive dashboard.

## 🎯 Objective
To perform end-to-end data analysis including data cleaning, transformation, analysis, and visualization, and to build a professional one-page dashboard.

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts  
- Slicers  
- Basic Excel Formulas  

## 🔄 Project Steps

### 1. Data Cleaning
- Removed duplicates and handled missing values  
- Standardized and formatted data  
- Used text functions for cleaning inconsistent data  

### 2. Data Transformation
- Applied formulas like SUM, AVERAGE, IF, COUNTIF  
- Used VLOOKUP, INDEX, MATCH for data retrieval  
- Created calculated columns  

### 3. Data Analysis
- Performed descriptive statistics  
- Identified patterns using filtering and sorting  
- Analyzed key metrics using pivot tables  

### 4. Data Visualization
- Created charts such as bar, pie, line, and area charts  
- Used slicers for interactivity  
- Designed a clean and structured dashboard layout  

## 📊 Key Insights
- India shows the highest contribution in salary distribution  
- Marketing and IT departments perform strongly compared to others  
- Salary trends vary across employees indicating performance differences  

## 📈 Final Output
- One-page interactive dashboard  
- Visual representation of key metrics  
- User-friendly and structured layout
